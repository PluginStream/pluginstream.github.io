PluginStream Install Guide

How to install

Pick your platform below — clear steps with exactly what to tap and what to allow, for both Android Phone and Android TV.

Recommended for You

Android Phone
1

Open the APK file

Open the APK file from your notification or Downloads folder. Most browsers will offer to open it instantly.

2

Tap "Settings"

When prompted, tap Settings and enable "Allow from this source" for your browser or file manager.

3

Tap Install

Go back and tap Install. Wait a few seconds while Android installs the package.

4

Tap Open

Launch PluginStream and enjoy! You can also find it in your app drawer afterwards.

Android TV
1

Transfer APK

Download the APK on Phone/PC and transfer it to your TV using a "Send Files to TV" app or a USB drive.

2

Open File Manager

On your TV, open any File Manager app (like X-plore or File Commander) and locate the APK.

3

Allow Installation

Select the APK. If prompted, go to Security settings and allow installation from unknown sources.

4

Complete Install

Follow the on-screen prompts to finish installation. PluginStream will appear in your Apps list.

⚠️
"Harmful file" warning? That's a false positive — PluginStream is not on the Play Store and is built from open-source code on GitHub. You can verify the SHA256 checksum on the release page.
