Before Contacting Us

To help us help you faster, please check these resources first:

  • README: Check our README file for container setup and repository instructions.
  • GitHub Issues: Search existing logs to see if your problem has already been reported.
  • Documentation: Check our custom setup guide for comprehensive instructions.

What to Include in Your Message

When writing to us, please make sure to supply:

  • Platform Information: Operating system version, device model (TV vs Phone).
  • App Version: The exact build version of PluginStream you're using.
  • Problem Description: Clear layout descriptions or crash traces.
  • Screenshots/Logs: Attach visual context or debug logs if applicable.

Response Times

I aim to evaluate all triage messages within:

  • Email: 24-48 hours response cycle.
  • GitHub Issues: 2-3 days average review.
  • Bug Reports: Priority tracking response within 24 hours.

Contributing

Want to contribute to PluginStream? We welcome repository contributions!

  • Code Contributions: Fork the repository and open custom pull requests.
  • Documentation: Help optimize ecosystem wiki pages and tutorial materials.
  • Extensions: Build or enhance modular streaming manifest files.
